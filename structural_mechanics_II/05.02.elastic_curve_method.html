교재 학습 범위

휨변형(bending deformation, flexural deformation)

  • 휨변형은 휨모멘트에 의해서 부재가 휘어지는 것으로서, 다음 그림과 같이 부재의 각 위치에서 아래쪽으로 처지는 처짐 \(y\), 원래 부재축으로부터 회전한 각도인 처짐각(회전각) \(\theta\)로 표현한다.

  • 휨변형은 일반적으로 부재의 각 위치마다 다른 값을 가지므로, 필요에 따라서는 부재의 모든 위치에서의 처짐과 처짐각을 계산해야 한다.
  • 확인 질문
    처짐 외에 휨변형을 정확하게 표현하기 위해서 계산해야 값을 쓰시오.

탄성곡선법(elastic curve method)의 원리

  • 다음 그림은 오직 휨모멘트 \(M\)만 작용하는 순수휨 상태의 부재이다. 여기에서 왼쪽 끝단을 원점으로, 오른쪽 방향을 \(+\)로 설정하여 부재의 위치를 \(x\)로 표현한다.

  • 위 부재에서 휨이 발생한 아주 작은 \(dx\)부분을 설명을 위해서 매우 과장하여 아래 그림에 나타냈다.

  • 여기에서, \(dx\)는 아주 작은 부분의 길이, \(\overline{ss}\)는 단면의 도심축(centroidal axis), \(ds\)는 도심축 \(\overline{ss}\)의 길이이다. \(y\)는 부재의 원래 위치에서 아래쪽으로 처진 처짐, \(\theta\)는 원래 부재축으로부터 회전한 각도인 처짐각이다. \(\rho\)는 회전반경, \(d\theta\)와 \(dy\)는 각각 \(dx\)부분의 시작점과 끝점의 처짐과 처짐각의 차이이다.
  • 위 그림에서 \(dx\)는 아주 작기 때문에 \(dx \approx ds, \, ds = \rho d\theta, \, \theta \approx \tan \theta = dy / dx \)이다. 따라서 다음이 성립한다. $$ dx \approx ds = \rho d\theta, \quad dx = \rho d\theta, \quad \frac{1}{\rho} = \frac{d\theta}{dx} \approx \frac{d \tan \theta}{dx} = \frac{d(dy / dx)}{dx} = \frac{d^2y}{dx^2} $$
  • 앞서 휨응력도 계산식을 유도하는 과정에서 얻은 식 \(\kappa = 1/\rho = - M / EI \)을 고려하면 다음 식을 얻을 수 있다. 여기에서 \(EI\)는 휨강성(flexural rigidity)이라 한다. $$ - \frac{M}{EI} = \frac{d\theta}{dx} = \frac{d^2y}{dx^2} $$
  • 위 식에서 \(M, \, \theta, \, y\)는 원점으로부터 오른쪽으로 \(x\)만큼 떨어진 위치의 휨모멘트, 처짐각, 처짐이므로, \(M(x), \, \theta(x), \, y(x)\)로 다시 표기하면 위의 식은 아래와 같이 된다. $$ - \frac{M(x)}{EI} = \frac{d\theta(x)}{dx} = \frac{d^2y(x)}{dx^2} $$
  • 위 식에서 부재의 원점으로부터 오른쪽으로 \(x\)만큼 떨어진 위치의 처짐각 \(\theta(x)\)와 처짐 \(y(x)\)를 계산하는 식은 다음과 같다. $$ \theta(x) = -\int \frac{M(x)}{EI}dx, \qquad y(x) = \int \theta(x)dx = -\iint \frac{M(x)}{EI}dx $$
  • 이를 보다 편리하게 적용학 수 있는 형태로 다시 쓰면 다음과 같다. $$ EI\theta(x) = -\int M(x)dx, \qquad EIy(x) = EI \int \theta(x)dx = -EI \iint M(x) dx $$

경계조건(boundary condition)

  • 휨변형을 계산할 때, 계산을 하지 않아도 특정한 부분에서의 처짐이나 처짐각을 바로 알 수 있다.
  • 지점은 부재의 이동이나 회전을 방지하는, 즉 이동이나 회전의 크기가 \(0\)이 되게 한다. 또한 절점에서 연결되는 부재들은 파괴되기 전에는 상대적인 위치와 회전각의 변화가 \(0\)이다.
  • 따라서 이러한 점을 이용하면 지점이나 특정한 절점에서 계산을 하지 않고도 휨변형, 즉 처짐이나 처짐각의 일부 미리 알 수 있다. 그리고 이를 경계조건(boundary condition)이라 한다.
  • 다음 그림과 같은 경우, 양쪽 지점에서의 처짐, 즉 \( y_A = y(0) = 0, \, y_B = y(L) = 0 \) 이다. 만약 하중이 수직축에 대해서 대칭으로 작용한다면 양쪽 지점에서의 처짐각, 즉 \(\theta_A = \theta(0) = - \theta_B = - \theta(L) \)이며, 가운데 에서의 처짐각 \(\theta(L/2) = 0\)이다.

  • 다음 그림과 같은 경우에는 고정단에서의 처짐각과 처짐, 즉 \( y_A = y(0) = 0, \, \theta_A = \theta(0) = 0 \) 이다.

  • 확인 질문
    등분포하중을 받는 단순보에서 양단 지점의 처짐 \(y\)를 계산하시오.
    \(mm\)

계산절차

  1. 구조물 전체를 대상으로 자유물체도를 작성하고 평형조건식을 적용하여 지점반력을 계산한다.
  2. 부재 중간에 작용하는 집중하중과 지점반력의 좌우로, 분포하중 작용구간과 그 좌우로 구간을 구분하고, 각 구간에 따라 부재의 위치를 표현하는 \(x\)의 원점과 양의 방향을 결정한다.
  3. 각 구간별로 휨모멘트 \(M(x)\)를 구한다.
  4. \(EI\theta(x) = -\int M(x)dx \)식으로 \(EI\theta(x)\)를 계산한다. 이때, 처짐각에 대한 경계조건을 적용하여 적분상수를 계산한다.
  5. \(EIy(x) = EI\int \theta(x)dx \)식으로 \(EIy(x)\)를 계산한다. 이때, 처짐에 대한 경계조건을 적용하여 적분상수를 계산한다.
  6. \(EI\theta(x), \, EIy(x) \)를 \(EI\)로 나누어 \(\theta(x), \, y(x)\)를 계산한다.
  7. 이 과정에서 다음 그림과 같이 처짐과 처짐각의 부호체계를 적용한다.

예제 1

  • 문제 : 탄성곡선법을 이용하여 처짐각과 처짐 \(\theta(x), \, y(x)\)를 계산하시오.

  • 풀이
    1. 다음과 같이 구조물 전체를 대상으로 자유물체도를 작성하고 평형조건식을 적용하여 지점반력을 계산한다.

    2. 위의 자유물체도에서 왼쪽 끝단에 지점반력이, 오른쪽 끝단에 외력이 집중하중으로 작용하므로 구간은 하나이다. 왼쪽 끝단을 원점으로, 오른쪽을 \(+\)로 부재의 위치를 나타내는 \(x\)를 설정하였다.
    3. 다음 그림과 같이 부재의 임의의 위치 \(x\)에서 가상으로 절단하여 자유물체도를 그리고 평형조건식을 적용하여 \(M(x)\)를 계산한다.

      \( \Sigma M_x = PL - Px + M(x) = 0; \quad M(x) = Px - PL \)
    4. 다음의 식과 같이 처짐각을 계산하되, 경계조건으로 \(\theta(0) = 0\)을 적용한다. $$ EI\theta(x) = -\int M(x)dx = \int (-Px + PL)dx = -\frac{Px^2}{2} + PLx + C_1 \\ EI\theta(0) = -\frac{P\times 0^2}{2} + PL\times 0 + C_1 = C_1 = 0; \quad C_1 = 0 \\ \therefore EI\theta(x) = -\frac{Px^2}{2} + PLx $$
    5. 다음의 식과 같이 처짐각을 계산하되, 경계조건으로 \(y(0) = 0\)을 적용한다. $$ EIy(x) = EI\int \theta(x)dx = \int \left( -\frac{Px^2}{2} + PLx \right)dx = -\frac{Px^3}{6} + \frac{PLx^2}{2} + C_2 \\ EIy(0) = -\frac{P\times 0^3}{6} + \frac{PL \times 0^2}{2} + C_2 = C_2 = 0; \quad C_2 = 0 \\ \therefore EIy(x) = -\frac{Px^3}{6} + \frac{PLx^2}{2} $$
    6. 위의 식들을 \(EI\)로 나누면 처짐각과 처짐은 다음과 같다. $$ \theta(x) = \frac{1}{EI} \left( -\frac{Px^2}{2} + PLx \right), \quad y(x) = \frac{1}{EI} \left( -\frac{Px^3}{6} + \frac{PLx^2}{2} \right) $$
    7. 자유단인 오른쪽 끝단의 처짐각과 처짐은 그 위치를 나타내는 \(x = L\)을 위의 식에 대입하여 다음과 같이 계산한다. $$ \theta(L) = \frac{1}{EI} \left( -\frac{PL^2}{2} + PL^2 \right) = \frac{PL^2}{2EI} \\ y(L) = \frac{1}{EI} \left( -\frac{PL^3}{6} + \frac{PL^3}{2} \right) = \frac{PL^3}{3EI} $$
    8. 처짐각과 처짐 모두 계산결과가 \(+\)이므로 처짐각은 시계방향, 처짐은 아래쪽인 것을 알 수 있다.

예제 2

  • 문제 : 탄성곡선법을 이용하여 처짐각과 처짐 \(\theta(x), \, y(x)\)를 계산하시오.

  • 풀이
    1. 다음과 같이 구조물 전체를 대상으로 자유물체도를 작성하고 평형조건식을 적용하여 지점반력을 계산한다.

    2. 위의 자유물체도에서 왼쪽 끝단에 지점반력인 집중하중이, 부재 전체에 외력인 분포하중이 작용하므로 구간은 하나이다. 왼쪽 끝단을 원점으로, 오른쪽을 \(+\)로 부재의 위치를 나타내는 \(x\)를 설정하였다.
    3. 다음 그림과 같이 부재의 임의의 위치 \(x\)에서 가상으로 절단하여 자유물체도를 그리고 평형조건식을 적용하여 \(M(x)\)를 계산한다.

      $$ \Sigma M_x = wL^2/2 - wLx + wx^2/2 + M(x) = 0; \\ M(x) = -wx^2/2 + wLx - wL^2/2 $$
    4. 다음의 식과 같이 처짐각을 계산하되, 경계조건으로 \(\theta(0) = 0\)을 적용한다. $$\begin{align} EI\theta(x) &= -\int M(x)dx = \int \left( \frac{wx^2}{2} - wLx + \frac{wL^2}{2} \right)dx \\ &= \frac{wx^3}{6} -\frac{wLx^2}{2} + \frac{wL^2x}{2} + C_1 \end{align}$$ $$ EI\theta(0) = \frac{w\times 0^3}{6} -\frac{wL \times 0^2}{2} + \frac{wL^2 \times 0}{2} + C_1 = C_1 = 0; \quad C_1 = 0 \\ \therefore EI\theta(x) = \frac{wx^3}{6} -\frac{wLx^2}{2} + \frac{wL^2x}{2} $$
    5. 다음의 식과 같이 처짐각을 계산하되, 경계조건으로 \(y(0) = 0\)을 적용한다. $$ \begin{align} EIy(x) &= EI\int \theta(x)dx = \int \left( \frac{wx^3}{6} -\frac{wLx^2}{2} + \frac{wL^2x}{2} \right)dx \\ &= \frac{wx^4}{24} -\frac{wLx^3}{6} + \frac{wL^2x^2}{4} + C_2 \end{align}$$ $$ EIy(0) = \frac{w\times 0^4}{24} -\frac{wL \times 0^3}{6} + \frac{wL^2 \times 0^2}{4} + C_2 = C_2 = 0; \quad C_2 = 0 \\ \therefore EIy(x) = \frac{wx^4}{24} -\frac{wLx^3}{6} + \frac{wL^2x^2}{4} $$
    6. 위의 식들을 \(EI\)로 나누면 처짐각과 처짐은 다음과 같다. $$ \theta(x) = \frac{1}{EI} \left( \frac{wx^3}{6} -\frac{wLx^2}{2} + \frac{wL^2x}{2} \right) \\ y(x) = \frac{1}{EI} \left( \frac{wx^4}{24} -\frac{wLx^3}{6} + \frac{wL^2x^2}{4} \right) $$
    7. 자유단인 오른쪽 끝단의 처짐각과 처짐은 그 위치를 나타내는 \(x = L\)을 위의 식에 대입하여 다음과 같이 계산한다. $$ \theta(L) = \frac{1}{EI} \left( \frac{wL^3}{6} -\frac{wL^3}{2} + \frac{wL^3}{2} \right) = \frac{wL^3}{6EI} \\ y(L) = \frac{1}{EI} \left( \frac{wL^4}{24} -\frac{wL^4}{6} + \frac{wL^4}{4} \right) = \frac{wL^4}{8EI} $$
    8. 처짐각과 처짐 모두 계산결과가 \(+\)이므로 처짐각은 시계방향, 처짐은 아래쪽인 것을 알 수 있다.

예제 3

  • 문제 : 탄성곡선법을 이용하여 처짐각과 처짐 \(\theta(x), \, y(x)\)를 계산하시오.

  • 풀이
    1. 다음과 같이 구조물 전체를 대상으로 자유물체도를 작성하고 평형조건식을 적용하여 지점반력을 계산한다.

    2. 위의 자유물체도에서 양쪽 끝단에 지점반력인 집중하중이, 부재 전체에 외력인 분포하중이 작용하므로 구간은 하나이다. 왼쪽 끝단을 원점으로, 오른쪽을 \(+\)로 부재의 위치를 나타내는 \(x\)를 설정하였다.
    3. 다음 그림과 같이 부재의 임의의 위치 \(x\)에서 가상으로 절단하여 자유물체도를 그리고 평형조건식을 적용하여 \(M(x)\)를 계산한다.

      $$ \Sigma M_x = -wLx/2 + wx^2/2 + M(x) = 0; \quad M(x) = -wx^2/2 + wLx/2 $$
    4. 다음의 식과 같이 처짐각을 계산하되, 경계조건으로 수직축에 대해 대칭인 것을 적용하여 \(\theta(L/2) = 0\)을 적용한다. $$\begin{align} EI\theta(x) &= -\int M(x)dx = \int \left( \frac{wx^2}{2} - \frac{wLx}{2} \right)dx \\ &= \frac{wx^3}{6} -\frac{wLx^2}{4} + C_1 \end{align}$$ $$ EI\theta(0) = \frac{w\times (L/2)^3}{6} -\frac{wL \times (L/2)^2}{4} + C_1 = -\frac{wL^3}{24} + C_1 = 0; \quad C_1 = \frac{wL^3}{24} \\ \therefore EI\theta(x) = \frac{wx^3}{6} -\frac{wLx^2}{4} + \frac{wL^3}{24} $$
    5. 다음의 식과 같이 처짐각을 계산하되, 경계조건으로 \(y(0) = 0\)을 적용한다. $$ \begin{align} EIy(x) &= EI\int \theta(x)dx = \int \left( \frac{wx^3}{6} -\frac{wLx^2}{4} + \frac{wL^3}{24} \right)dx \\ &= \frac{wx^4}{24} -\frac{wLx^3}{12} + \frac{wL^3x}{24} + C_2 \end{align}$$ $$ EIy(0) = \frac{w\times 0^4}{24} -\frac{wL \times 0^3}{12} + \frac{wL^3 \times 0}{24} + C_2 = C_2 = 0; \quad C_2 = 0 \\ \therefore EIy(x) = \frac{wx^4}{24} -\frac{wLx^3}{12} + \frac{wL^3x}{24} $$
    6. 위의 식들을 \(EI\)로 나누면 처짐각과 처짐은 다음과 같다. $$ \theta(x) = \frac{1}{EI} \left( \frac{wx^3}{6} -\frac{wLx^2}{4} + \frac{wL^3}{24} \right) \\ y(x) = \frac{1}{EI} \left( \frac{wx^4}{24} -\frac{wLx^3}{12} + \frac{wL^3x}{24} \right) $$
    7. 왼쪽 끝단과 오른쩍 끝단의 처짐각은 그 위치를 나타내는 \(x=0, \, x=L\)을 처짐각 식에 대입하여 다음과 같이 계산한다. $$ \theta(0) = \frac{1}{EI} \left( \frac{w\times 0^3}{6} -\frac{wL \times 0^2}{4} + \frac{wL^3}{24} \right) = \frac{wL^3}{24EI} \\ \theta(L) = \frac{1}{EI} \left( \frac{wL^3}{6} -\frac{wL^3}{4} + \frac{wL^3}{24} \right) = -\frac{wL^3}{24EI} $$
    8. 부재의 가운데에서 발생하는 처짐은 그 위치를 나타내는 \(x = L/2\)을 처짐 식에 대입하여 다음과 같이 계산한다. $$\begin{align} y(L/2) &= \frac{1}{EI} \left( \frac{w(L/2)^4}{24} - \frac{wL(L/2)^3}{12} + \frac{wL^3(L/2)}{24} \right) \\ &= \frac{1}{EI} \left( \frac{wL^4}{24\times16} - \frac{wL^4}{12\times8} + \frac{wL^4}{24\times2} \right) \\ &= \frac{1}{EI} \frac{wL^4}{24\times2} \left( \frac{1}{8} - \frac{1}{2} + \frac{1}{1} \right) = \frac{1}{EI} \frac{wL^4}{24\times2} \left( \frac{1}{8} - \frac{4}{8} + \frac{8}{8} \right) \\ &= \frac{1}{EI} \frac{wL^4}{24\times2} \left( \frac{1-4+8}{8} \right) = \frac{1}{EI} \frac{wL^4}{24\times2} \frac{5}{8} \\ &= \frac{5wL^4}{384EI} \end{align}$$ $$ \therefore y_{\max} = \frac{5wL^4}{384EI} $$
    9. 이상에서 \(\theta(0) > 0, \, \theta(L) < 0\)이므로 왼쪽 끝단은 처짐각이 시계방향, 오른쪽 끝단은 반시계방향인 것을 알 수 있다. 그리고 \(y(L/2) > 0\)이므로 처짐은 아래쪽인 것을 알 수 있다.

예제 4

  • 문제 : 탄성곡선법을 이용하여 처짐각과 처짐 \(\theta(x), \, y(x)\)를 계산하시오.

  • 풀이
    1. 다음과 같이 구조물 전체를 대상으로 자유물체도를 작성하고 평형조건식을 적용하여 지점반력을 계산한다.

    2. 위의 자유물체도에서 양쪽 끝단에 지점반력인 집중하중이, 부재 중간에 외력인 집중하중이 작용하므로 구간은 2개이다. 원칙적으로 2개의 구간에 대해서 처짐각과 처짐을 계산해야 하지만, 수직축에 대해 대칭인 점을 고려하여 왼쪽의 첫번째 구간에 대해서만 처짐각과 처짐을 계산한다. 따라서 왼쪽 끝단을 원점으로, 오른쪽을 \(+\)로 부재의 위치를 나타내는 \(x\)를 설정하였다.
    3. 다음 그림과 같이 부재의 임의의 위치 \(x\)에서 가상으로 절단하여 자유물체도를 그리고 평형조건식을 적용하여 \(M(x)\)를 계산한다.

      $$ \Sigma M_x = -Px/2 + M(x) = 0; \quad M(x) = Px/2 $$
    4. 다음의 식과 같이 처짐각을 계산하되, 경계조건으로 수직축에 대해 대칭인 것을 적용하여 \(\theta(L/2) = 0\)을 적용한다. $$ EI\theta(x) = -\int M(x)dx = \int \left( -\frac{Px}{2} \right)dx = -\frac{Px^2}{4} + C_1 \\ EI\theta(0) = -\frac{P \times (L/2)^2}{4} + C_1 = -\frac{PL^2}{16} + C_1 = 0; \quad C_1 = \frac{PL^2}{16} \\ \therefore EI\theta(x) = -\frac{PLx^2}{4} + \frac{PL^2}{16} $$
    5. 다음의 식과 같이 처짐각을 계산하되, 경계조건으로 \(y(0) = 0\)을 적용한다. $$ EIy(x) = EI\int \theta(x)dx = \int \left( -\frac{PLx^2}{4} + \frac{PL^2}{16} \right)dx = -\frac{Px^3}{12} + \frac{PL^2x}{16} + C_2 $$ $$ EIy(0) = -\frac{P \times 0^3}{12} + \frac{PL^2 \times 0}{16} + C_2 = C_2 = 0; \quad C_2 = 0 \\ \therefore EIy(x) = -\frac{Px^3}{12} + \frac{PL^2x}{16} $$
    6. 위의 식들을 \(EI\)로 나누면 처짐각과 처짐은 다음과 같다. $$ \theta(x) = \frac{1}{EI} \left( -\frac{PLx^2}{4} + \frac{PL^2}{16} \right) \\ y(x) = \frac{1}{EI} \left( -\frac{Px^3}{12} + \frac{PL^2x}{16} \right) $$
    7. 왼쪽 끝단의 처짐각은 그 위치를 나타내는 \(x=0\)을 처짐각 식에 대입하여 다음과 같이 계산한다. $$ \theta(0) = \frac{1}{EI} \left( -\frac{PL\times 0^2}{4} + \frac{PL^2}{16} \right) = \frac{PL^2}{16EI} $$ 여기에서 \(\theta(0) > 0\)이므로 처짐각은 시계방향인 것을 알 수 있다. 오른쪽 끝단의 처짐각은 수직축에 대칭이므로 왼쪽 끝단의 처짐각과 크기는 같으며, 방향은 반대인 반시계방향이다.
    8. 부재의 가운데에서 발생하는 처짐은 그 위치를 나타내는 \(x = L/2\)을 처짐 식에 대입하여 다음과 같이 계산한다. $$\begin{align} y(L/2) &= \frac{1}{EI} \left( -\frac{P(L/2)^3}{12} + \frac{PL^2(L/2)}{16} \right) \\ &= \frac{1}{EI} \left( -\frac{PL^2}{12\times8} + \frac{PL^3}{16\times2} \right) \\ &= \frac{1}{EI} \frac{PL^3}{16\times2} \left( -\frac{1}{3} + \frac{1}{1} \right) = \frac{1}{EI} \frac{PL^3}{16\times2} \left( \frac{2}{3} \right) = \frac{PL^3}{48EI} \end{align}$$ 여기에서 \(y(L/2) > 0\)이므로 처짐은 아래쪽인 것을 알 수 있다.

예제 5

  • 문제 : 탄성곡선법을 이용하여 처짐각과 처짐 \(\theta(x), \, y(x)\)를 계산하시오.

  • 풀이
    1. 다음과 같이 구조물 전체를 대상으로 자유물체도를 작성하고 평형조건식을 적용하여 지점반력을 계산한다.

    2. 위의 자유물체도에서 양쪽 끝단에 지점반력인 집중하중이, 부재내에 외력인 집중하중이 작용하므로 구간은 2개이다. 먼저 \(0 \le x \le a\) 구간에 대해서 처짐각과 처짐을 계산하고 이어서 \(a \le x \le (a+b) = L\) 구간에 대해서 처짐각과 처짐을 계산한다. 두 구간 모두 왼쪽 끝단을 원점으로, 오른쪽을 \(+\)로 부재의 위치를 나타내는 \(x\)를 설정하였다.
    3. 다음 그림과 같이 \(0 \le x \le a\) 구간에 대해서 부재의 임의의 위치 \(x\)에서 가상으로 절단하여 자유물체도를 그리고 평형조건식을 적용하여 \(M(x)\)를 계산한다.

      $$ \Sigma M_x = -Pbx/L + M(x) = 0; \quad M(x) = Pbx/L $$
    4. 다음의 식과 같이 처짐각과 처짐을 계산한다. $$ EI\theta_1(x) = -\int M(x)dx = \int \left( -\frac{Pbx}{L} \right)dx = -\frac{Pbx^2}{2L} + C_1 \\ EIy_1(x) = EI\int \theta_1(x)dx = \int \left( -\frac{Pbx^2}{2L} + C_1 \right)dx = -\frac{Pbx^3}{6L} + C_1x + C_2 $$
    5. 다음 그림과 같이 \(a \le x \le L\) 구간에 대해서 부재의 임의의 위치 \(x\)에서 가상으로 절단하여 자유물체도를 그리고 평형조건식을 적용하여 \(M(x)\)를 계산한다.

      $$ \Sigma M_x = -Pbx/L + P(x-a) + M(x) = 0; \quad M(x) = Pbx/L - P(x-a) $$
    6. 다음의 식과 같이 처짐각과 처짐을 계산한다. $$ EI\theta_2(x) = -\int M(x)dx = \int \left( -\frac{Pbx}{L} + P(x-a) \right)dx = -\frac{Pbx^2}{2L} + \frac{P(x-a)^2}{2} + C_3 \\ EIy_2(x) = EI\int \theta_2(x)dx = \int \left( -\frac{Pbx^2}{2L} + \frac{P(x-a)^2}{2} + C_3 \right)dx = -\frac{Pbx^3}{6L} + \frac{P(x-a)^3}{6} + C_3x + C_4 $$
    7. 이 문제에서는 경계조건으로 먼저 양 끝단 처짐의 크기가 \(0\), 즉 \(EIy_1(0) = 0, \, EIy_2(L) = 0 \)을 적용할 수 있다. 그리고 이에 더하여 경계조건으로 집중하중이 가해지는 위치에서 두 개 구간의 처짐각과 처짐의 크기가 같다는 것, 즉 \(EI\theta_1(a) = EI\theta_2(a), \, EIy_1(a) = EIy_2(a)\)을 적용할 수 있다. 적용한 내용은 다음과 같다. $$ EIy_1(0) = -\frac{Pb\times0^3}{6L} + C_1\times0 + C_2 = C_2 = 0 \\ EIy_2(L) = -\frac{PbL^3}{6L} + \frac{P(L-a)^3}{6} + C_3L + C_4 = -\frac{PbL^2}{6} + \frac{Pb^3}{6} + C_3L + C_4 = 0 \\ EI\theta_1(a) = -\frac{Pba^2}{2L} + C_1 = EI\theta_2(a) = -\frac{Pba^2}{2L} + \frac{P(a-a)^2}{2} + C_3 = -\frac{Pba^2}{2L} + C_3 \\ EIy_1(a) = -\frac{Pba^3}{6L} + C_1a + C_2 = EIy_2(a) = -\frac{Pba^3}{6L} + \frac{P(a-a)^3}{2} + C_3a + C_4= -\frac{Pba^3}{6L} + C_3a + C_4 $$
    8. 첫번째 식에서, 세번째 식에서, 네번째 식에서, 마지막으로 두번째 식에서 다음의 결과를 차례로 알 수 있다. $$ C_2 = 0, \quad C_1 = C_3, \quad C_4 = C_2 = 0, \quad C_1 = C_3 = \frac{Pb(L^2-b^2)}{6L} $$
    9. 따라서 \( C_1, \, C_2, \, C_3, \, C_4 \)를 대입하면 두 구간의 처짐각과 처짐은 다음과 같다. $$ EI\theta_1(x) = -\frac{Pbx^2}{2L} + \frac{Pb(L^2-b^2)}{6L} \\ EIy_1(x) = -\frac{Pbx^3}{6L} + \frac{Pbx(L^2-b^2)}{6L} \\ EI\theta_2(x) = -\frac{Pbx^2}{2L} + \frac{P(x-a)^2}{2} + \frac{Pb(L^2-b^2)}{6L} \\ EIy_2(x) = -\frac{Pbx^3}{6L} + \frac{P(x-a)^3}{6} + \frac{Pbx(L^2-b^2)}{6L} $$
    10. 위의 식들을 \(EI\)로 나누면 처짐각과 처짐은 다음과 같다. $$ \theta_1(x) = \frac{1}{EI} \left( -\frac{Pbx^2}{2L} + \frac{Pb(L^2-b^2)}{6L} \right) \\ y_1(x) = \frac{1}{EI} \left( -\frac{Pbx^3}{6L} + \frac{Pbx(L^2-b^2)}{6L} \right) \\ \theta_2(x) = \frac{1}{EI} \left( -\frac{Pbx^2}{2L} + \frac{P(x-a)^2}{2} + \frac{Pb(L^2-b^2)}{6L} \right) \\ y_2(x) = \frac{1}{EI} \left( -\frac{Pbx^3}{6L} + \frac{P(x-a)^3}{6} + \frac{Pbx(L^2-b^2)}{6L} \right) $$
